# 🚀 Design and Analysis of Algorithms (DAA) Lab Simulators

A collection of **Design and Analysis of Algorithms (DAA)** laboratory experiments implemented in **Python** along with **interactive web-based algorithm simulators** built using **HTML, CSS, and JavaScript**.

Unlike traditional lab repositories that only contain source code, this repository provides interactive visualizations that help users understand algorithm execution, compare performance, and explore algorithm behavior in real time.

---

## 📌 Repository Features

* 🐍 Python implementations for all DAA experiments
* 🌐 Interactive web simulators for each experiment
* 📊 Performance comparison between algorithms
* 🎯 Real-time algorithm visualization
* 📈 Execution time benchmarking
* 📉 Charts and performance analytics
* 💻 Responsive and modern UI
* 🌙 Dark/Light mode support
* ⚡ Runs completely offline

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/DAA-Lab.git
```

### Run Python Programs

```bash
python experiment1.py
```

### Open the Simulator

Simply open the corresponding `simulator.html` file in any modern web browser.

No installation or internet connection is required.
